 OKX Trading Bot - 설치 안내
================================================

❌ "QFluentWidgets" 패키지 오류가 발생했나요?
✅ 걱정하지 마세요! 자동으로 수정할 수 있습니다.


📌 해결 방법 (간단!)
================================================

이 폴더에서 다음 파일을 더블클릭하세요:

    👉 0_fix_installation_error.bat

그러면 자동으로:
  ✓ 기존 설치 삭제
  ✓ 올바른 패키지로 재설치
  ✓ 모든 오류 수정

약 5~10분 정도 소요됩니다.


📌 완전히 처음 설치하는 경우
================================================

순서대로 더블클릭하세요:

1. 1_install_python_and_deps.bat
   → Python 설치 (이미 있으면 건너뜀)
   → 완료 후 PC 재부팅 권장

2. 0_fix_installation_error.bat
   → 가상환경 생성 및 패키지 설치
   → 5~10분 소요

3. 3_run_app.bat
   → 앱 실행!


📌 이미 Python이 설치되어 있는 경우
================================================

바로 이것만 더블클릭하세요:

    👉 0_fix_installation_error.bat


📌 중요 사항
================================================

✓ Python 버전: 3.8 ~ 3.13 필요 (권장: 3.11)
✓ 인터넷 연결 필요
✓ 설치 중에는 창을 닫지 마세요

❌ Python 3.14 이상은 지원하지 않습니다
❌ Python 3.7 이하는 지원하지 않습니다
